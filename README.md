# BondRemainder

*Bond RemainderTracker Dashboard*

Track when you last contacted friends, set relationship goals, and log interactions.

## Technologies

- Next.js 14 (App Router)
- React
- Tailwind CSS
- Recharts
- React Icons
- React Hot Toast

## Key Features

- *Friend Dashboard* – View all friends with status indicators (overdue, almost due, on-track).
- *Interaction Timeline* – Log calls, texts, and video chats; filter and sort history.
- *Friendship Analytics* – Pie chart showing interaction type distribution.
- *Fully Responsive* – Works on mobile, tablet, and desktop.
- *Persistent Storage* – Timeline entries saved in browser localStorage.
